# Capacitor

## Que es
Un capacitor almacena energia en un campo electrico entre dos placas.

## Fisica
- Acumula carga electrica y genera diferencia de potencial.
- En DC ideal, en regimen permanente se comporta como circuito abierto.

## Formulas
- `Q = C * V`
- `i(t) = C * dv(t)/dt`
- `E = 0.5 * C * V^2`
- `Xc = 1 / (2 * pi * f * C)`

## Variaciones
- Ceramico, electrolitico, tantalio, film.
- ESR y fuga no ideales.
- Polarizados y no polarizados.
